# G.U.N.D.A.M
ROS package for hobby humanoid robot


## Settup

Flash Dietpi (32 bit) onto the SD card 
Set the wifi connection in the text file
Connect with SSH
Enable bluetooth on first bootup or later through `dietpi-config`
Install python3 and pip3
